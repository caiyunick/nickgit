1. 分支管理
      多人协助
    • 删除远程仓库命令：git remote rm 远程仓库名
    • 查看远仓信息：git remote/git remote -v
    • 本地推送分只：git push orgin branch-name
    • 抓取远程新提交：git pull
    • 本地创建远程对应的分只：git checkout -b branch-name orgin/branch-name
    • 本地分只和远程分只关联：git branch --set-upstream branch-name orgin/branch-name
    • 多人开发
  2. 标签管理
    • 创建标签：git tag v1.0( commitID)
    • 查看所有标签：git tag
    • 查看标签说明：git show v1.0
    • 本地删除标签：git tag -d v1.0
    • 远程删除标签：git tag origin :refs/tags/v1.0
    • 推送标签到远程：git push origin v1.0/--tags
